This repository contains all the data and code used in the manuscript entitled 'Exploring the abiotic and biotic drivers of uneven species richness in salamanders and newts' All code is written in the R language (R Core Development Team, 2022). 
Folders:
- code: fully chaptered and annotated code used in the study.
- data: data used to perform all analyses.
    * bd_models_data: data used in the birth-death models
    * trees: trees and subtrees used in the different analyses
- outputs: outputs of the different analyses.
    * ase: ancestral state estimation for all caudata, plethodontidae and salamandridae
    * bd_models_outputs: birth-death models outputs (DATA NOT AVAILABLE ON GITHUB REPOSITORY DUE TO FILE SIZE LIMITATIONS)
    * pgls_family_level: data used to test for the influence of macroevolutionary rates on species richness at the family level
    * pgls_genus_level: data used to test for the influence of macroevolutionary rates on species richness at the genus level
    * run_bamm: outputs from bamm diversification rate analysis
    * run_clads: outputs from clads diversification rate analysis (DATA NOT AVAILABLE ON GITHUB REPOSITORY DUE TO FILE SIZE LIMITATIONS)
    * test_corr_bamm_clads: data used to test for correlation between macroevolutionary rates estimated from bamm and clads
